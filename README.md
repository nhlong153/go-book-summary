# BOOK SUMMARY USING CHATGPT TEXT GENERATION MODEL

## HOW TO USE:

<ol>
    <li>Step 1: Open book_summary.go under the cmd directory, fill in the secret key to connect to OpenAI. Then build the file to executable using go build </li>
    <li>Step 2: Add the file contains the text you want to summary into the execute directory</li>
    <li>Step 3: Run ./book_summary "fileName" (replace your file name into "fileName")</li>
</ol>
